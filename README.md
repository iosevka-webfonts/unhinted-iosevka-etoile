# Unhinted-iosevka-etoile WebFont 17.1.0

## How to use

- Add `<link href="https://iosevkawebfonts.github.io/Unhinted-iosevka-etoile/Unhinted-iosevka-etoile.css" rel="stylesheet" />` to your `<head>`.
- Use `fontFamily: 'Unhinted iosevka etoile Web'` or `font-family: 'Unhinted iosevka etoile Web'`.
